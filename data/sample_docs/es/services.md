# Clínica Dental BrightSmile - Servicios

## Odontología general
Revisiones rutinarias, limpieza profesional, empastes del color del diente y
extracciones. Recomendamos una revisión cada seis meses.

## Ortodoncia
Opciones para alinear los dientes, incluidos los frenillos tradicionales y los
alineadores transparentes. El tratamiento comienza con una consulta de
ortodoncia para evaluar la idoneidad.

## Blanqueamiento dental
Blanqueamiento profesional en la clínica y kits para casa. La idoneidad para
este servicio cosmético se confirma mediante una breve evaluación.

## Implantes dentales
Sustitución de dientes ausentes con implantes dentales duraderos. El tratamiento
comienza con una consulta de implantes y pruebas de imagen.

## Atención dental de urgencia
Ofrecemos evaluaciones de urgencia el mismo día para dolor, hinchazón, sangrado o
traumatismos dentales. Contacta directamente con la clínica ante una urgencia.
Este asistente no puede diagnosticar ni tratar afecciones médicas; solo se
registra una solicitud de revisión humana cuando una acción completada lo
confirma.
